# View JuteVision inside Cursor

1. **Start the app** (if not running):  
   In terminal run:  
   `.\venv\Scripts\python.exe -m streamlit run jute_test.py --server.port 8501`

2. **Open the app in Cursor:**
   - Press **Ctrl+Shift+P**
   - Type **Simple Browser: Show**
   - Press **Enter**
   - Type **http://localhost:8501** and press **Enter**

The app will open in a browser tab **inside Cursor**.

---

**Alternative:** Open `preview_app.html` in the editor, then right‑click → **Open with Live Server** (if you have the extension). Or use **Simple Browser: Show** and enter:  
`file:///C:/Users/AFTAB/Desktop/New%20folder/JuteVision/preview_app.html`  
to see the app in an embedded frame.
